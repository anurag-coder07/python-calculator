# Python Calculator

A command-line calculator built with Python. It supports common arithmetic operations, percentage calculations, and exponent values.

## Features

- addition
- subtraction
- multiplication
- division
- percentage calculation
- power calculation
- input validation
- division by zero handling
- cleaner result formatting
- option to clear the screen and start again

## Run

```bash
python calculator.py
```

## Tests

```bash
python -m unittest discover -s tests
```
