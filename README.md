# Professional Python Calculator

A clean and beginner-friendly calculator project built with Python. This application performs basic arithmetic operations through a simple command-line interface and is structured to look polished on GitHub.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input validation for numbers and operations
- Protection against division by zero

## Project Structure

```text
.
|-- calculator.py
|-- README.md
`-- .gitignore
```

## Requirements

- Python 3.8 or higher

## How to Run

1. Open a terminal in the project folder.
2. Run the following command:

```bash
python calculator.py
```
